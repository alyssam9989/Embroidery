# Embroidery
This program is intended to log embroidery projects as well as track the cost and hours that went into them. 

# Inputs
## Name
## Goal Hours for the Week
## Menu
1. Add Embroidery Session.
2. View Sessions.
3. Get Recommendation.
4. Save Report.
5. Quit.

# Output
1. Description.
2. Hours.
3. Cost.
4. Difficulty.
