# Embroidery
## This program is intended to track embroidery projects, as well as the hours spent and cost associated with them. 

# Inputs
 ## Name.
 ## Weekly goal for embroidery hours.
 
 ## Menu 
 1. Add embroidery session.
 2. View sessions.
 3. Get recommendation.
 4. Save report.
 5. Quit.

# Outputs
 1. Description
 2. Hours
 3. Cost
 4. Difficulty
